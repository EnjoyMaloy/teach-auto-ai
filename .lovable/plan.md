
## Добавление гайда по получению ссылки на шрифт из Google Fonts

### Что нужно сделать

Добавить пошаговую инструкцию в компонент `CustomFontInput`, которая объясняет как получить ссылку на шрифт с Google Fonts.

### Изменения

**Файл: `src/components/editor/design-system/CustomFontInput.tsx`**

Добавлю сворачиваемый блок с гайдом под полем ввода. Гайд будет содержать:

1. **Три способа добавления шрифта:**
   - Просто ввести название (например: `Montserrat`)
   - Вставить ссылку со страницы шрифта
   - Вставить CSS-ссылку

2. **Пошаговая инструкция:**
   - Открыть fonts.google.com
   - Найти нужный шрифт
   - Скопировать URL из адресной строки
   - Вставить в поле ввода

3. **Примеры ссылок:**
   - `Montserrat`
   - `https://fonts.google.com/specimen/Montserrat`

### Технические детали

- Добавлю иконку `HelpCircle` для переключения видимости гайда
- Гайд будет скрыт по умолчанию, чтобы не перегружать интерфейс
- Используем `Collapsible` компонент из Radix UI для плавного раскрытия
- Ссылка на fonts.google.com будет кликабельной и откроется в новой вкладке
