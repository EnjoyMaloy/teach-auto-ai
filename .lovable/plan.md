
# План: Профессиональная UI-структура на основе референсов

## Философия подхода

Как верно указано в посте — ИИ отлично заполняет детали, но плохо создаёт структуру. План использует проверенные UI-паттерны от профессиональных дизайнеров (Tailwind UI, Shadcn Blocks) как каркас.

---

## 1. Унификация визуального стиля

### Проблема
Главная страница (Home) использует тёмную тему с анимированными blob-градиентами, а Catalog/Dashboard/Favorites — светлую. Это создаёт ощущение разных продуктов.

### Решение
Распространить тёмную тему с фиолетовыми акцентами на все страницы приложения.

**Изменения:**
- `src/pages/Catalog.tsx` — тёмный фон `bg-[#0f0f12]`, белый текст
- `src/pages/Favorites.tsx` — аналогичный стиль
- `src/pages/Dashboard.tsx` — адаптация под тёмную тему
- `src/components/catalog/CourseCard.tsx` — тёмная карточка с подсветкой
- `src/components/catalog/CategoryFilter.tsx` — категории с прозрачными фонами

## 3. Улучшенные карточки курсов

### Паттерн: "Card with subtle glow on hover"

**Текущее состояние:**
- Базовый hover с shadow

**Новый стиль:**
- Тёмный фон `bg-[#1a1a1b]`
- Тонкая граница `border-white/5`
- При hover: фиолетовое свечение `shadow-[0_0_30px_rgba(168,85,247,0.15)]`
- Плавная анимация категории

**Файл:** `src/components/catalog/CourseCard.tsx`

### Новые CSS-классы (в index.css):

```css
/* Карточка с glow-эффектом */
.card-glow {
  @apply bg-[#1a1a1b] border border-white/5 rounded-2xl;
  @apply transition-all duration-300;
}
.card-glow:hover {
  @apply border-primary/30;
  box-shadow: 0 0 30px rgba(168, 85, 247, 0.15);
}

/* Hero секция */
.hero-gradient {
  background: radial-gradient(ellipse 80% 50% at 50% -20%, 
    hsl(270 40% 25% / 0.5), transparent);
}
```

### Токены дизайн-системы (уже существуют):

- Фон: `#0f0f12` (Background Dark)
- Карточки: `#1a1a1b` (Card Dark)
- Акцент: `hsl(265, 60%, 75%)` (Accent Name)
- Primary: `hsl(243, 75%, 58%)`
