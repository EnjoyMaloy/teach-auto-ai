

## Редизайн выбора модели иллюстраций

### Проблема
Сейчас при выборе "С картинками" снизу появляется второй ряд из двух кнопок с названиями моделей — это выглядит неинтуитивно и перегружает интерфейс. Пользователю непонятно, зачем ему знать название модели.

### Решение
Заменить переключатель "С картинками / Быстро" + отдельный ряд моделей на **один ряд из трёх кнопок** с понятными человеческими названиями:

```text
ИЛЛЮСТРАЦИИ
[ ★ Качество ] [ ⚡ Быстро ] [ Без картинок ]
```

- **Качество** — включает картинки, модель Gemini 3 Pro (иконка Star)
- **Быстро** — включает картинки, модель Gemini 2.5 Flash (иконка Zap)
- **Без картинок** — отключает генерацию картинок (иконка ImageOff)

Так пользователь делает выбор за один клик, а не за два. Названия моделей скрыты — вместо них понятные характеристики.

### Технические детали

**Файл: `src/components/editor/EditorAISidebar.tsx`**

1. Удалить текущие два блока (строки 522-574): переключатель "С картинками / Быстро" и условный блок выбора модели
2. Вставить вместо них один ряд из трёх кнопок в `flex gap-1.5`
3. Логика клика:
   - "Качество": `setLocalSkipImages(false)`, `setImageModel('gemini-3-pro')`
   - "Быстро": `setLocalSkipImages(false)`, `setImageModel('gemini-2.5-flash')`
   - "Без картинок": `setLocalSkipImages(true)`
4. Активное состояние определяется комбинацией `localSkipImages` и `imageModel`
5. Импортировать `Zap` и `Star` из `lucide-react`
